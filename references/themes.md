## 使用说明

1. **选择模板**：根据活动类型选择对应 Style (A-F) 的 HTML 模板文件
2. **选择配色**：从该 Style 下的 3 个变体中选择最匹配的预设
3. **替换变量**：将选中的 `:root { ... }` 块直接替换模板文件顶部的同名块
4. **填充内容**：从 `<!-- POSTER_CONTENT -->` 标记处开始替换为实际活动信息
